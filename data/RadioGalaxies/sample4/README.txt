Data extracted from Radio Galaxy Zoo (class galaxy C2) taken from FIRST survey.  
Original data taken from https://cloudstor.aarnet.edu.au/plus/s/agKNekOJK87hOh0.    
See also https://github.com/chenwuperth/rgz_rcnn/issues/10 in CLARAN github repo.

Bounding boxes, originally provided in the RGZ storage in XML format, were converted to DS9 and stored in regions_bbox dir. 
Raw masks were produced with Caesar finder, selecting only sources extracted inside the bounding box.
Data were inspected one by one. When the cutout contains more than 1 object (besides the radio galaxy), a region was prepared and made available in dir regions/. 
These regions were used to create an updated mask. 
After the selection, these images were not considered for training (input images FITS still available in the imgs dir but no mask provided for them):
